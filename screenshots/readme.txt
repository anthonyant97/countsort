Count Sort Simulation - Read Me

- Range input : 0 - 20
- Minimum total number to start count sort : 2
- Number only
- Start button : begin count sort
- Delete button : delete a number from list, select a number first
- Reset button : back to the original state, only available after input number or start count sort
- Next button : proceed to the next step
- Prev button : back to the previous step
- First button : back to the first step, only available after click next button
